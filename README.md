# Family Grocery Planner

A small static web app for planning weekly family dinners, grocery deliveries, pantry use, and recipe ideas.

## Files

- `index.html` is the app structure.
- `styles.css` controls the layout and visual design.
- `app.js` contains the interactive planner behavior.

## Open It

Open `index.html` directly in a browser. There is no build step.

If Node.js is installed, you can also run a small static server:

```powershell
npm run dev
```

Then open `http://localhost:5173`.

The original copied output is still preserved in `outputs/family-grocery-planner`.
